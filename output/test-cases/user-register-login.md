# 用户注册与登录 - 测试用例

## 1 正向流程测试

### 1.1 用户使用有效手机号注册成功

- **前置条件**：手机号未被注册
- **步骤**：
  1. 进入注册页面
  2. 输入手机号 13800138000
  3. 点击获取验证码
  4. 输入收到的6位验证码
  5. 设置密码 Test@1234
  6. 点击注册
- **预期结果**：注册成功，跳转到首页，显示欢迎信息

### 1.2 用户使用手机号+密码登录成功

- **前置条件**：用户已注册账号
- **步骤**：
  1. 进入登录页面
  2. 选择"密码登录"标签
  3. 输入已注册手机号 13800138000
  4. 输入正确密码 Test@1234
  5. 点击登录
- **预期结果**：登录成功，跳转到首页，显示用户信息

### 1.3 用户使用验证码登录成功

- **前置条件**：用户已注册账号
- **步骤**：
  1. 进入登录页面
  2. 选择"验证码登录"标签
  3. 输入已注册手机号 13800138000
  4. 点击获取验证码
  5. 输入收到的6位验证码
  6. 点击登录
- **预期结果**：登录成功，跳转到首页，显示用户信息

## 2 异常流程测试

### 2.1 使用已注册手机号再次注册

- **前置条件**：手机号 13800138000 已被注册
- **步骤**：
  1. 进入注册页面
  2. 输入手机号 13800138000
  3. 点击获取验证码
- **预期结果**：提示"该手机号已注册，请直接登录"，提供登录入口链接

### 2.2 使用未注册手机号登录

- **前置条件**：手机号未注册
- **步骤**：
  1. 进入登录页面
  2. 输入手机号 13900139000
  3. 输入密码 Test@1234
  4. 点击登录
- **预期结果**：提示"该手机号未注册"，引导用户注册

### 2.3 使用错误密码登录

- **前置条件**：用户已注册，密码为 Test@1234
- **步骤**：
  1. 进入登录页面
  2. 输入手机号 13800138000
  3. 输入错误密码 Wrong@123
  4. 点击登录
- **预期结果**：提示"密码错误，请重新输入"

### 2.4 输入错误验证码注册

- **前置条件**：手机号未注册
- **步骤**：
  1. 进入注册页面
  2. 输入手机号 13800138000
  3. 点击获取验证码
  4. 输入错误验证码 123456
  5. 设置密码 Test@1234
  6. 点击注册
- **预期结果**：提示"验证码错误，请重新输入"

### 2.5 验证码过期后注册

- **前置条件**：手机号未注册
- **步骤**：
  1. 进入注册页面
  2. 输入手机号 13800138000
  3. 点击获取验证码
  4. 等待验证码过期（5分钟后）
  5. 输入已过期的验证码
  6. 设置密码 Test@1234
  7. 点击注册
- **预期结果**：提示"验证码已过期，请重新获取"

## 3 边界值测试

### 3.1 密码长度恰好为最小值

- **前置条件**：注册页面
- **步骤**：输入密码 Aa1@5678（恰好8位）
- **预期结果**：密码校验通过

### 3.2 密码长度低于最小值

- **前置条件**：注册页面
- **步骤**：输入密码 Aa1@567（仅7位）
- **预期结果**：提示"密码长度不能少于8位"

### 3.3 密码长度恰好为最大值

- **前置条件**：注册页面
- **步骤**：输入密码 Aa1@567890123456（恰好20位）
- **预期结果**：密码校验通过

### 3.4 密码长度超过最大值

- **前置条件**：注册页面
- **步骤**：输入密码 Aa1@5678901234567（21位）
- **预期结果**：提示"密码长度不能超过20位"

### 3.5 手机号位数不足

- **前置条件**：注册页面
- **步骤**：输入手机号 1380013800（仅10位）
- **预期结果**：提示"请输入正确的手机号格式"

### 3.6 手机号位数过多

- **前置条件**：注册页面
- **步骤**：输入手机号 138001380000（12位）
- **预期结果**：提示"请输入正确的手机号格式"

## 4 输入校验测试

### 4.1 手机号输入包含字母

- **步骤**：在手机号输入框输入 138abc00000
- **预期结果**：提示"请输入正确的手机号格式"

### 4.2 手机号输入包含特殊字符

- **步骤**：在手机号输入框输入 138-0013-8000
- **预期结果**：提示"请输入正确的手机号格式"

### 4.3 密码不包含数字

- **步骤**：输入密码 Aa@abcdef
- **预期结果**：提示"密码需包含数字、大小写字母和特殊字符"

### 4.4 密码不包含大写字母

- **步骤**：输入密码 aa1@abcdef
- **预期结果**：提示"密码需包含数字、大小写字母和特殊字符"

### 4.5 密码不包含小写字母

- **步骤**：输入密码 AA1@ABCDEF
- **预期结果**：提示"密码需包含数字、大小写字母和特殊字符"

### 4.6 密码不包含特殊字符

- **步骤**：输入密码 Aa1abcdef
- **预期结果**：提示"密码需包含数字、大小写字母和特殊字符"

### 4.7 密码输入框插入 XSS 脚本

- **步骤**：在密码输入框输入 `<script>alert('xss')</script>`
- **预期结果**：脚本被转义或过滤，不执行，正常保存为纯文本

### 4.8 手机号输入框为空

- **步骤**：不输入手机号，直接点击获取验证码
- **预期结果**：提示"请输入手机号"

### 4.9 密码输入框为空

- **步骤**：不输入密码，直接点击登录
- **预期结果**：提示"请输入密码"

## 5 状态流转测试

### 5.1 连续登录失败后账号被锁定

- **前置条件**：用户已注册，密码为 Test@1234
- **步骤**：
  1. 连续输入错误密码 5 次
  2. 第6次输入正确密码登录
- **预期结果**：提示"账号已被锁定，请30分钟后再试"

### 5.2 锁定后等待解锁时间再登录

- **前置条件**：账号已被临时锁定
- **步骤**：
  1. 等待30分钟
  2. 输入正确密码登录
- **预期结果**：登录成功

## 6 幂等性测试

### 6.1 快速连续点击注册按钮

- **前置条件**：手机号未注册
- **步骤**：
  1. 输入手机号 13800138000
  2. 点击获取验证码
  3. 输入验证码 123456
  4. 设置密码 Test@1234
  5. 快速连续点击注册按钮两次
- **预期结果**：只注册成功一次，不会重复创建账号

### 6.2 快速连续点击获取验证码

- **前置条件**：手机号未注册
- **步骤**：
  1. 输入手机号 13800138000
  2. 快速连续点击获取验证码按钮多次
- **预期结果**：只发送一次验证码，提示"验证码已发送"
