# 在线教育平台 - 课程购买与学习模块

## 背景与目标

为了提升用户学习体验，我们需要开发一个在线教育平台的课程购买与学习模块。用户可以浏览课程、购买课程、在线学习，并获得学习证书。

## 用户注册与登录

### 注册流程

#### 功能描述
用户通过手机号注册账号。输入手机号后点击获取验证码，输入验证码后设置密码完成注册。

#### 输入与输出
- **输入**：手机号、验证码、密码
- **输出**：注册成功/失败结果

#### 业务规则
1. 手机号仅支持中国大陆手机号（1开头，11位数字）
2. 每个手机号只能注册一个账号
3. 验证码有效期为5分钟
4. 验证码每个手机号每天最多发送10次
5. 密码最小长度为8位，最大长度为20位
6. 密码必须包含大小写字母和数字
7. 注册成功后系统立即发送欢迎邮件

#### 约束条件
- 手机号格式：1[3-9]\d{9}
- 验证码长度：6位数字
- 验证码有效期：300秒（5分钟）
- 验证码每日发送上限：10次/手机号
- 密码最小长度：8位
- 密码最大长度：20位
- 密码复杂度：必须包含大写字母、小写字母、数字

### 登录流程

#### 功能描述
用户可以使用手机号+密码登录，也可以使用验证码登录。

#### 业务规则
1. 手机号+密码登录：输入手机号和密码验证
2. 验证码登录：输入手机号，获取验证码，输入验证码登录
3. 连续登录失败5次后，账号被临时锁定30分钟
4. 锁定期间用户无法登录，可通过忘记密码重置或等待自动解锁

#### 异常处理
- 登录失败提示：用户名或密码错误
- 账号锁定提示：账号已被锁定，请30分钟后重试或使用忘记密码

#### 约束条件
- 登录失败锁定阈值：5次连续失败
- 锁定时长：30分钟

## 课程浏览

### 功能描述
用户可以按分类浏览课程列表，查看课程详情，进行搜索和排序。

#### 业务规则
1. 课程列表展示课程封面、标题、讲师、价格、评分等信息
2. 支持按分类浏览课程
3. 支持按价格（升序/降序）、销量（降序）、评分（降序）排序
4. 支持关键词搜索，搜索结果分页展示

#### 约束条件
- 每页返回结果数：20条
- 搜索结果最大返回：200条
- 搜索响应时间：≤1秒
- 页面加载时间：≤3秒

#### 接口性能要求
- 接口响应时间：≤1秒
- 搜索响应时间：≤1秒
- 视频播放启动时间：≤2秒

### 课程详情页

#### 功能描述
展示课程的详细信息，包括大纲、讲师介绍、用户评价、相关课程推荐等。

#### 业务规则
1. 课程大纲：展示所有章节和课时列表
2. 讲师介绍：展示讲师姓名、头像、简介
3. 用户评价：展示已购买用户的评价和评分
4. 相关课程推荐：推荐同分类或同讲师的课程

## 课程购买

### 购买流程

#### 功能描述
用户在课程详情页点击"立即购买"，进入订单确认页，选择支付方式进行支付。

#### 业务规则
1. 订单确认页展示课程信息、价格、优惠抵扣金额
2. 支持使用优惠券，每个订单只能使用一张
3. 支持微信支付和支付宝支付
4. 支付成功后订单状态变更为已支付

#### 异常处理
- 支付失败：提示用户支付失败原因，订单保持待支付状态，用户可重新支付
- 支付超时：支付超时时间为15分钟，超时后订单自动变更为已取消

#### 约束条件
- 支付超时时间：15分钟

### 订单管理

#### 订单状态流转

| 状态 | 说明 | 可转换状态 | 转换条件 |
|------|------|------------|----------|
| 待支付 | 订单已创建，等待用户支付 | 已支付、已取消 | 支付成功回调 / 用户主动取消 / 支付超时自动取消 |
| 已支付 | 用户已成功支付 | 已完成、已取消、已退款 | 课程学习完成 / 用户申请退款且审核通过 |
| 已取消 | 订单已被取消 | 不可恢复 | 用户主动取消 / 支付超时自动取消 / 退款成功 |
| 已完成 | 课程学习完成 | 不可变更 | 观看完所有章节视频 |
| 已退款 | 已支付订单退款成功 | 不可恢复 | 退款审核通过 |

#### 订单取消
- 未支付订单：用户可在创建后15分钟内主动取消
- 已支付订单：用户可在支付后7天内申请退款，且课程学习进度不超过20%

#### 退款流程
1. 退款条件：支付后7天内可申请退款，课程学习进度不超过20%
2. 退款申请需要审核，审核时效：1-3个工作日
3. 退款退回原支付渠道
4. 退款成功后订单状态变更为已退款

#### 约束条件
- 未支付订单取消时限：15分钟
- 已支付订单退款时限：支付后7天内
- 退款申请审核时效：1-3个工作日

### 优惠券

#### 功能描述
用户可以使用优惠券抵扣部分金额。

#### 业务规则
1. 每个订单只能使用一张优惠券
2. 优惠券有使用门槛和有效期
3. 优惠券门槛：订单金额满100元可用
4. 优惠券有效期：自发放之日起30天
5. 优惠券金额：固定金额（如10元、20元、50元）或折扣（如9折、8折）

#### 约束条件
- 优惠券使用门槛：满100元
- 优惠券有效期：30天

## 在线学习

### 视频播放

#### 功能描述
用户购买课程后可以在线观看课程视频。

#### 业务规则
1. 支持多种清晰度切换：480P、720P、1080P
2. 支持倍速播放：0.5x、1x、1.25x、1.5x、2x
3. 记录用户的学习进度，下次打开时从上次位置继续播放
4. 进度记录频率：每10秒自动保存一次

#### 约束条件
- 视频清晰度选项：480P、720P、1080P
- 视频倍速选项：0.5x、1x、1.25x、1.5x、2x
- 进度记录频率：10秒

### 学习记录

#### 功能描述
系统记录用户的学习时长和完成进度。

#### 业务规则
1. 课程学习完成的判定标准：观看完所有章节的视频
2. 系统实时记录学习时长
3. 系统实时记录每个课时的观看进度

#### 约束条件
- 进度记录持久化：每次观看每10秒记录一次

### 课程评价

#### 功能描述
用户完成课程后可以对课程进行评价。

#### 业务规则
1. 评价包括评分（1-5星）和文字评论
2. 评分必填，文字评论选填
3. 评价发布后不可修改或删除
4. 评价无需审核，直接显示
5. 文字评论字数限制：50-500字

#### 约束条件
- 评分范围：1-5星
- 文字评论字数：50-500字

## 学习证书

### 功能描述
用户完成课程学习后可以获得电子证书。

#### 业务规则
1. 证书包含：用户姓名、课程名称、完成日期
2. 证书格式：PDF
3. 课程学习完成后自动生成证书
4. 用户可下载PDF证书
5. 用户可打印证书

#### 管理员权限
1. 可手动颁发证书
2. 可撤销证书
3. 可查看所有证书

#### 约束条件
- 证书格式：PDF
- 证书内容：用户姓名、课程名称、完成日期

### 性能要求

- 页面首次加载时间：≤3秒
- 接口响应时间：≤1秒
- 搜索响应时间：≤1秒
- 视频播放启动时间：≤2秒

### 安全性要求

- 密码存储：MD5加盐加密
- 支付安全：遵循PCI DSS标准
- 敏感数据脱敏：手机号中间4位用*代替，如138****1234
- 验证码：单次有效，限次使用，防暴力破解

### 兼容性要求

- 桌面端浏览器：Chrome、Firefox、Safari、Edge 最新两个版本
- 移动端iOS：iOS 12+
- 移动端Android：Android 8+
- 支持设备：手机、平板
