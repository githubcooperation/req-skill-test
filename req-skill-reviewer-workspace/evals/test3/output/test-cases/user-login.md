## 用例列表

### TC-UL-001：使用有效手机号和密码成功登录（正向流程）

- **类型**：正向流程
- **优先级**：P0
- **前置条件**：手机号 13800138000 已注册，密码为 Test1234
- **测试步骤**：
  1. 打开登录页面
  2. 在手机号输入框中输入 13800138000
  3. 在密码输入框中输入 Test1234
  4. 点击"登录"按钮
- **预期结果**：
  - 登录成功
  - 页面跳转至首页
  - 用户处于已登录状态
- **关联需求**：用户登录 > 账号密码登录

---

### TC-UL-002：使用验证码方式成功登录（正向流程）

- **类型**：正向流程
- **优先级**：P0
- **前置条件**：手机号 13800138001 已注册；可接收短信验证码
- **测试步骤**：
  1. 打开登录页面
  2. 切换至"验证码登录"选项卡
  3. 输入手机号 13800138001
  4. 点击"获取验证码"按钮
  5. 等待接收短信，输入收到的6位验证码
  6. 点击"登录"按钮
- **预期结果**：
  - 登录成功
  - 跳转至首页
  - 用户已登录
- **关联需求**：用户登录 > 验证码登录
