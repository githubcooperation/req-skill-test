## 用例列表

### TC-UR-001：使用有效手机号和验证码完成注册（正向流程）

- **类型**：正向流程
- **优先级**：P0
- **前置条件**：手机号 13800138001 未被注册；网络连接正常
- **测试步骤**：
  1. 打开注册页面
  2. 在手机号输入框中输入 13800138001
  3. 点击"获取验证码"按钮
  4. 等待接收短信，在验证码输入框中输入收到的6位数字（如 123456）
  5. 设置登录密码 Test1234
  6. 点击"注册"按钮
- **预期结果**：
  - 页面显示"注册成功"提示
  - 系统自动登录用户
  - 3秒后自动跳转至首页
  - 用户已登录状态保持
- **关联需求**：用户注册 > 注册流程 > 标准注册步骤
