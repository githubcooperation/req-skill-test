## 1. 项目背景

本模块为 EduSaaS 平台的课程搜索功能。用户可通过关键词搜索课程、按分类筛选、查看课程详情。所有功能均为纯前端展示，数据由已有的课程列表 API 提供（内部系统，非新增集成接口）。

---

## 2. 业务流程

用户在平台首页通过搜索框或分类导航找到感兴趣的课程，点击进入详情页了解课程信息后决定是否购买。整个流程无跨系统调用。

---

## 3. 功能需求

### 3.1 课程搜索

用户在搜索框输入关键词，系统展示匹配的课程列表。

**业务规则**：
- 搜索范围：课程名称、讲师名称、课程描述（取前50字）
- 搜索结果按相关度排序，无结果时展示空状态页
- 搜索词少于 2 个字符时不触发搜索，展示提示"请输入至少2个字"

**页面字段及逻辑描述**：

| 字段名 | 是否必需 | 字段说明 |
|--------|----------|----------|
| 搜索关键词 | 是 | 2-50个字符 |

**错误处理**：
- 搜索服务不可用时展示"搜索暂时不可用，请稍后重试"

### 3.2 分类筛选

用户可通过左侧分类树筛选课程。

**业务规则**：
- 支持多级分类（最多3级）
- 选择子分类时父分类自动高亮
- 取消所有筛选条件后恢复展示全部课程

### 3.3 课程详情

用户点击课程卡片进入详情页。

**页面字段及逻辑描述**：

| 字段名 | 是否必需 | 字段说明 |
|--------|----------|----------|
| 课程名称 | 是 | 展示完整课程名 |
| 讲师名称 | 是 | — |
| 课程简介 | 否 | 最多 500 字 |
| 价格 | 是 | 单位：元，保留2位小数 |
| 课程封面图 | 否 | 无图时展示占位图 |

**业务规则**：
- 未登录用户可查看课程详情，但点击"立即购买"跳转至登录页
