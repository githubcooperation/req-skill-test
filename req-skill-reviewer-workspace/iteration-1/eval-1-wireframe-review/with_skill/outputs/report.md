## req-wireframe/SKILL.md 专项审查报告

### 亮点

1. **输入文件优先级链清晰**（B1）：明确了 `output/prd-updated.md` 优先，不存在时主动提示用户选择，降级策略有话术模板。
2. **占位符定位逻辑细致**（B2/C1）：用"三元组"（模块名、需求卡片名、屏幕名称）定义每个占位符上下文，追溯逻辑按层级向上，规则清晰且可执行。
3. **文件命名规则有示例**（C3）：给出了具体示例，降低了 LLM 自由发挥的空间。
4. **CSS 变量完整内联**（C4）：在 SKILL.md 内直接嵌入了完整的 `:root` 变量块，无需 LLM 自行查找 `wireframe-style.html`。
5. **截图容错机制完备**（B4）：两次失败后给出明确的降级话术，不因单个失败卡死流程。
6. **已有文件 A/B/C/D 菜单**（B3）：四选项完整，覆盖全量覆盖、增量生成、指定重生成、取消四种意图。
7. **七条结构化自检项**，帮助 LLM 在写完每个文件后自我核验。
