## 优先级分层方法

用 **价值 × 频率 × 可行性** 粗排，研发一起确认后调整。

- P0：MVP 必须有，上线即需要
- P1：上线后第一个迭代补充
- P2：中期规划，视资源推进
- P3：长期 / 待定 / 需要专项评估

---

## P0 · 核心功能（MVP）

### 任务基础能力
- [ ] 创建 / 编辑 / 删除任务
- [ ] 设置负责人、截止日期、优先级
- [ ] 任务状态流转（待开始 → 进行中 → 已完成 → 已关闭）
- [ ] 子任务（至少支持两级）

### 视图
- [ ] 列表视图（默认）
- [ ] "我的任务"视图（今日待办 / 即将到期 / 全部）

### 通知
- [ ] 任务分配时通知被分配人
- [ ] 截止日期前 N 小时提醒（N 可配置）
- [ ] 任务状态变更通知相关人

### 权限
- [ ] 项目级权限隔离（不同项目互不可见）
- [ ] 角色：项目管理员 / 成员 / 只读

---

## P1 · 第一迭代

### 协作增强
- [ ] 任务评论 / @提及
- [ ] 附件上传（图片、文档）
- [ ] 任务动态日志（谁在什么时间做了什么操作）

### 模板
- [ ] 任务模板（单任务 + 子任务结构）
- [ ] 项目模板（一键复制整个项目结构）

### 多维过滤 & 搜索
- [ ] 按标签/负责人/截止日期/状态过滤
- [ ] 全局搜索

### 移动端
- [ ] H5 自适应（先不做原生 App）
- [ ] 移动端基础功能：查看、更新状态、评论

---

## P2 · 中期规划

### 任务关系
- [ ] 任务依赖（前置任务完成才能开始）
- [ ] 关联任务（相关但无依赖）

### 数据统计
- [ ] 任务完成率统计
- [ ] 平均处理时长（对客服场景有价值）
- [ ] 个人/团队工作量报表

### 甘特图视图
- [ ] 时间轴视图（销售/运营项目管理场景）

### 重复任务
- [ ] 按日/周/月自动创建重复任务

---

## P3 · 长期 / 待评估

- [ ] CRM 系统集成（需要确认接口，单独立项）
- [ ] 工单系统集成（客服场景）
- [ ] 飞书 SSO / 企微 SSO 对接
- [ ] AI 辅助：自动识别任务优先级、智能提醒
- [ ] 数据导出（Excel / CSV）**← 研发建议 P1，我觉得可以，待确认**
